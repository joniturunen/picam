<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

- Raspberry Pi (4 tested)
- Camera module (v.2.1)
- Raspberry Pi power brick or battery pack
- MicroSD card for the OS (of your choice)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/joniturunen/picam.git
   ```
2. Install NPM packages
   ```sh
   pip install -r requirements.txt
   ```
